## Session 3: ‡πÄ‡∏£‡∏µ‡∏¢‡∏Å‡πÉ‡∏ä‡πâ‡∏á‡∏≤‡∏ô‡∏Ç‡πâ‡∏≠‡∏°‡∏π‡∏•‡∏†‡∏≤‡∏û‡∏ñ‡πà‡∏≤‡∏¢‡∏î‡∏≤‡∏ß‡πÄ‡∏ó‡∏µ‡∏¢‡∏° 

### ‡∏≠‡∏ò‡∏¥‡∏ö‡∏≤‡∏¢
‡∏Å‡∏≤‡∏£‡πÄ‡∏£‡∏µ‡∏¢‡∏Å ImageCollection ‡∏Ç‡∏≠‡∏á Sentinel-2 ‡∏´‡∏£‡∏∑‡∏≠ Landsat ‡πÄ‡∏û‡∏∑‡πà‡∏≠‡∏ô‡∏≥‡∏°‡∏≤‡∏õ‡∏£‡∏∞‡∏°‡∏ß‡∏•‡∏ú‡∏• ‡πÇ‡∏î‡∏¢‡∏°‡∏µ‡∏´‡∏±‡∏ß‡∏Ç‡πâ‡∏≠‡∏¢‡πà‡∏≠‡∏¢‡∏î‡∏±‡∏á‡∏ô‡∏µ‡πâ:
- filterDate()
- filterBounds()
- filter ‡∏î‡πâ‡∏ß‡∏¢ CLOUD_PERCENTAGE
- composite ‡πÅ‡∏ö‡∏ö median
- Visualization parameter

### ‡∏ï‡∏±‡∏ß‡∏≠‡∏¢‡πà‡∏≤‡∏á‡πÇ‡∏Ñ‡πâ‡∏î
```javascript

  .filter(ee.Filter.lt('CLOUDY_PIXEL_PERCENTAGE', 10))
  .median();

Map.centerObject(roi, 10);
Map.addLayer(s2, vis, 'Sentinel-2');
```
